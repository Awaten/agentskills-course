## 🚀 路徑一：快速入門（建議新手）

適合「我想先搞清楚這是什麼東西」的人。

| 順序 | 類型 | 標題 | 時間 |
|------|------|------|------|
| 1 | 📖 秘笈 S1 | [Agent Skills 是什麼？](秘笈/01-what-are-agent-skills) | 5 min |
| 2 | 📖 秘笈 S2 | [SKILL.md 格式 3 分鐘](秘笈/02-skills-md-format) | 5 min |
| 3 | 📖 秘笈 S3 | [Description 觸發優化](秘笈/03-description-optimization) | 8 min |

**完成後** → 你已經知道 Agent Skills 的核心概念，可以直接試寫第一個 Skill

---

## 🔧 路徑二：實戰派（建議有寫程式經驗）

適合「少廢話，讓我直接動手做」的人。

| 順序 | 類型 | 標題 | 時間 |
|------|------|------|------|
| 1 | 📖 秘笈 S5 | [從零到發布 — 骰子 Skill 實戰](秘笈/05-zero-to-publish) | 10 min |
| 2 | 📖 秘笈 S4 | [7 個高效 Instructions 套路](秘笈/04-instruction-patterns) | 8 min |
| 3 | 📚 課程 Ch4 | [從零建置第一個 Skill（完整版）](課程/02-01-quickstart) | 20 min |

**完成後** → 你已經寫過一個完整 Skill，可以開始優化自己的專案

---

## 🎓 路徑三：系統學習（建議打造個人品牌）

適合「我要從零學到能開課/接案/帶團隊」的人。

| 階段 | 章節 | 內容 | 預計時間 |
|------|------|------|---------|
| **基礎** | Ch1-3 | 什麼是 Skills、三層揭露、vs Tools/MCP | 1 小時 |
| **實作** | Ch4-6 | 建置 Skill、SKILL.md 完整參考、跨平台測試 | 2 小時 |
| **心法** | Ch7-10 | 最佳實務、Description 優化、Scripts、大型架構 | 2.5 小時 |
| **驗證** | Ch11-12 | Eval 系統設計、評估與迭代 | 1.5 小時 |
| **生態** | Ch13-15 | 平台生態、API/MCP 整合、三層架構 | 2 小時 |
| **發布** | Ch16-17 | 發布分享、社群貢獻 | 1 小時 |
| **總計** | | **17 章完整學習** | **10-12 小時** |

## 哪條路適合你？

| 你的狀況 | 建議路徑 |
|---------|---------|
| 完全沒聽過 Agent Skills | 路徑一：快速入門 |
| 已經用過 GPT/Claude，想讓 AI 更聽話 | 路徑一 → 路徑二 |
| 會寫程式，想自己建 Agent 技能 | 路徑二：實戰派 |
| 想成為團隊中的 Agent Skills 專家 | 路徑三：系統學習 |
| 想做 Agent 相關創業/接案 | 路徑三：全部學完 |
