## 🚀 部署與開發

### [Vercel](https://vercel.com/)
> **免費部署你的 Agent Skill 網站**
- 靜態網站一鍵部署，支援 GitHub 自動整合
- 免費方案每天 6000 分鐘建置額度，個人或小型團隊綽綽有餘
- 原生支援 VitePress、Next.js 等框架
- 這個網站就是部署在 Vercel 上的

### [Supabase](https://supabase.com/)
> **開源 Firebase 替代方案**
- PostgreSQL 資料庫 + 即時 API + 認證系統
- 免費方案含 500MB 資料庫，適合 Skill 的後端整合
- 如果你要讓 Skill 儲存長期記憶或使用者偏好，Supabase 是最佳選擇

## 🤖 AI Agent 平台

### [Claude Code](https://claude.ai/)
> **最強的 AI Coding Agent**
- Anthropic 開發，原生支援 Agent Skills（SKILL.md）
- Claude Pro $20/月，Claude Max $100-200/月
- 直接在終端機中讀寫檔案、跑測試、部署

### [Cursor](https://cursor.com/)
> **AI-first 程式編輯器**
- 內建 Agent Skills 支援（.cursor/skills/）
- 支援 Claude、GPT-4o 等多種模型
- Free 版可用，Pro 版 $20/月

### [N8N](https://n8n.io/)
> **無程式碼 AI Agent 工作流平台**
- 視覺化拖放介面，不需寫程式就能建立自動化工作流
- 內建 AI Agent 節點，支援 Claude、GPT-4o
- 免費版可自託管，Cloud $20/月起
- 適合非技術背景使用者
